# Owlkettle Enhance Config (Default Titlebar)

The parser reads the first `jsonc` fenced block.

```jsonc
{
  // Keep native window title bar.
  "disable_titlebar": false,

  // If true, app-level stylesheet is loaded.
  "use_theme_stylesheet": true,

  // Fixes controls stretching to full width/height.
  "disable_stretching_boxes": true
}
```
